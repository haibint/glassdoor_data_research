tutorial 1: Background Knowledge and introduction

by Haibin TANG

Posted on July 15, 2019 at 12:00 AM

Welcome to this series of Web Scrapying Tutorial Series. In this Series, we are going to teach you how to write short computer programes to capture information from internet automatically for all kinds of Research. This is the very first episode the the series and let's talk about some prerequisites and set up.

This tutorial Series assumes attendance to have minimal programming skills in Python. If you have never done any Python programming before, I recommend This Youtube Playlist to get yourself familiar with Basic Python Programing.

However, if the following code below looks familiar to you, you are probably good to go.

# Some Basic Python skills that you should know

# output to command line
print("Hello world")

# for loop
for character in "Hello":
print(character)

# simple mathmatical operations
print(2*2)
print(2-2)
print(2+2)

# list and indexing
a_random_list = [10, 29, 384, 21]
print(a_random_list[2])

In this Tutorial, I will be using Python version 3.7.3 with MacOS 10.14.5, but any version of Python above 3.5 should work just fine and if you are using a PC, you probably want to install A software called GitBash in order to follow along.
